# 📈 Stock Portfolio Tracker

A user-friendly **Gradio-powered Python web app** to manage, visualize, and track stock portfolios using **real-time stock market data**.

---

## 🚀 Features

- ➕ Add new stocks with symbol, number of shares, and purchase price
- ➖ Remove stocks from your portfolio
- 📂 View your current portfolio in a clean format
- 📊 Track real-time total value using live stock prices via yFinance
- 📉 Visualize historical stock price trends (1-month chart)

---

## 🛠️ Technologies Used

- Python 3.x
- [Gradio](https://www.gradio.app/)
- [yFinance](https://pypi.org/project/yfinance/)
- Pandas, Matplotlib
